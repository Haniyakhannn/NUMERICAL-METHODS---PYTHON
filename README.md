# Numerical Methods Using Python – Graphical Analysis

## Project Description

This project implements various **Numerical Methods using Python** to solve problems related to numerical integration and differential equations. The solutions are also represented graphically using plots for better visualization and analysis.

## Numerical Integration Methods

The following methods were applied:

* Trapezoidal Rule
* Simpson’s 1/3 Rule
* Simpson’s 3/8 Rule

These methods were used to evaluate different integrals, and the area under each curve was visualized using graphs.

## Differential Equation Methods

The following numerical techniques were implemented:

* Modified Euler Method
* Runge-Kutta 4th Order (RK4) Method

These methods were used to solve first-order differential equations with given initial conditions and step sizes.

## Technologies Used

* Python
* NumPy
* Matplotlib

## Features

* Numerical approximation of integrals
* Solution of ordinary differential equations
* Graphical visualization of functions
* Shaded area under curves
* Comparison of numerical methods

## Conclusion

This project demonstrates the practical application of numerical methods using Python and helps in understanding mathematical problem-solving through computation and graphical analysis.
